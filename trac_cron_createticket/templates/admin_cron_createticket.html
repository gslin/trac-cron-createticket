Template Variables

You can use the following placeholders in title, description, and priority. All times are in UTC.

  • [now] - Current UTC datetime
  • [now_unix] - Current Unix timestamp
  • [today] - Today's date in UTC (YYYY-MM-DD)
  • [tomorrow] - Tomorrow's date in UTC (YYYY-MM-DD)
  • [yesterday] - Yesterday's date in UTC (YYYY-MM-DD)
  • [offset:N] - Date N seconds from now in UTC (e.g., [offset:86400] for tomorrow)
